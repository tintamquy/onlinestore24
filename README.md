# Thần Nông Store - Capstone Project (Online Store 24)

Dự án Capstone xây dựng website thương mại điện tử chuyên cung cấp sản phẩm thảo dược và nông sản thiên nhiên.

## 🚀 Tính năng nổi bật

### Dành cho Khách hàng
- Xem danh sách sản phẩm, lọc và phân trang.
- Đăng nhập / Đăng ký tài khoản.
- Mua hàng (Thêm/Sửa/Xóa giỏ hàng).
- Đặt hàng với cơ chế nhận diện qua số điện thoại.
- Đánh giá sản phẩm (Rating 1-5 sao) và viết Review.
- Xem lịch sử mua hàng.

### Dành cho Admin (Quản trị viên)
- Quản lý Sản phẩm (Thêm/Sửa/Xóa).
- Quản lý Khách hàng (Thêm/Sửa/Xóa).
- Quản lý Đơn hàng (Cập nhật trạng thái, Thay đổi chi tiết đơn hàng).
- Quản lý Đánh giá (Xóa các review vi phạm).
- Xem báo cáo Doanh thu và Khách hàng VIP bằng biểu đồ Chart.js.
- Xuất file báo cáo ra định dạng Excel (SheetJS).

## 🛠️ Công nghệ sử dụng
- **Frontend:** HTML5, CSS3, Vanilla JS, Bootstrap 5, AdminLTE 3.2, Chart.js, SheetJS, Reveal.js.
- **Backend:** Java Spring Boot, Spring Security, Hibernate.
- **Database:** MySQL.
- **Triển khai:** Render (Docker).
