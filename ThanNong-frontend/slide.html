Đồ án Capstone: Cửa hàng Trực tuyến

THẦN NÔNG STORE

Nền tảng thương mại điện tử chuyên cung cấp Thảo Dược & Nông Sản Thiên Nhiên.


Học viên: TRẦN XUÂN QUÝ

1. Tổng quan & Mục tiêu

  • Mục tiêu: Xây dựng hoàn chỉnh một ứng dụng Web Full-stack phục vụ việc bán hàng trực tuyến với trải nghiệm mượt mà.
  • Đối tượng:
    • Khách hàng: Mua sắm, đánh giá, xem lịch sử mua hàng.
    • Quản trị viên (Admin): Quản lý Sản phẩm, Khách hàng, Đơn hàng, Thống kê doanh thu.

2. Công nghệ sử dụng

Frontend

  • HTML5, CSS3, Vanilla JS
  • Bootstrap 5
  • AdminLTE 3.2 (Dashboard)
  • Chart.js & SheetJS

Backend

  • Java Spring Boot
  • Spring Security & JWT
  • Hibernate / Spring Data JPA
  • MySQL (Aiven Cloud)

3. Chức năng Dành cho Khách hàng

  • 🛍️ Khám phá, lọc và tìm kiếm sản phẩm.
  • 🛒 Quản lý Giỏ hàng thông minh (Thêm/Sửa/Xóa).
  • 📝 Đặt hàng (Checkout) với cơ chế tự động nhận diện và cập nhật thông tin qua Số Điện Thoại.
  • ⭐ Đánh giá và viết Review sản phẩm.
  • 📦 Xem lại Lịch sử Đơn hàng và trạng thái đơn.

4. Chức năng Dành cho Admin

  • Sản phẩm: Thêm, Sửa, Xóa, Upload Ảnh.
  • Khách hàng: Quản lý danh sách, Xem lịch sử mua hàng, Xóa khách hàng.
  • Đơn hàng: Tạo đơn hàng nội bộ, Cập nhật chi tiết Sản phẩm trong đơn, Đổi trạng thái.
  • Thống kê: Báo cáo Doanh thu, Báo cáo Khách hàng VIP. Vẽ biểu đồ và Xuất file Excel.
  • Kiểm duyệt: Quản lý và Xóa các Đánh giá vi phạm.

5. Cấu trúc Database (ERD)

Tuân thủ cấu trúc chuẩn, gồm 9 bảng chính:

  • products, product_lines: Quản lý mặt hàng.
  • customers: Quản lý khách hàng.
  • orders, order_details: Hệ thống đặt hàng.
  • reviews: Đánh giá sản phẩm.
  • users, roles: Định danh và phân quyền hệ thống.
  • payments, employees, offices: (Tích hợp mở rộng).
